## API Connection & Status

###`GET /api/status`
Retrieves the current connection status of the backend API (e.g., OpenAI or other LLM provider).

*   **Response:**
```json
    {
      "connected": true,
      "server_ok": true,
      "model": "gpt-4",
      "url_configured": true,
      "key_configured": true,
      "model_configured": true
    }
    ```
    *   On error, returns`connected: false` with`error`,`error_type` (e.g.,`auth_failed`,`config_missing`), and`action` fields.

###`POST /api/reconnect`
Attempts to reconnect to the configured backend API.

*   **Response:**
```json
    {
      "success": true
    }
    ```

###`POST /api/disconnect`
Disconnects the current backend API connection.

*   **Response:**
```json
    {
      "success": true
    }
    ```

###`GET /api/models`
Lists available models from the connected API.

*   **Response:**
```json
    {
      "models": [
        { "id": "gpt-4", "owned_by": "openai" },
        { "id": "gpt-3.5-turbo", "owned_by": "openai" }
      ]
    }
    ```
*   **Errors:** Returns`503` if not connected,`500` on backend error.

## Messaging

###`GET /messages`
Retrieves all messages in the current chat context.

*   **Response:**
```json
    {
      "messages": [
        { "role": "user", "content": "Hello", "index": 0, ... }
      ],
      "count": 1,
      "current_chat_id": "uuid-string"
    }
    ```

###`GET /messages/since`
Retrieves messages starting from a specific index (used for polling new messages).

*   **Query Parameters:**
    *`index` (int): The message index to start from.

*   **Response:**
```json
    {
      "messages": [...],
      "count": 1,
      "total": 5,
      "current_chat_id": "uuid",
      "current_chat_title": "Chat Title",
      "current_chat_tags": ["tag1"]
    }
    ```

###`POST /send`
Sends a message to the backend and waits for the full response.

*   **Request Body:** JSON object containing message data (structure depends on backend implementation).
*   **Response:**
```json
    {
      "response": { ... },
      "total": 2,
      "current_chat": {
        "id": "uuid",
        "title": "New Chat"
      }
    }
    ```
*   **Errors:** Returns`503` if API not connected.

###`POST /stream`
Initiates a Server-Sent Events (SSE) stream for AI responses.

*   **Request Body:** JSON object containing message data.
*   **Response:**`text/event-stream`
    *   **Event Data:**
        *   Start:`data: {"id": "stream_uuid"}`
        *   Token:`data: {"type": "content", "text": "..."}`
        *   Error:`data: {"error": true, "error_data": {...}}`
        *   Cancelled:`data: {"cancelled": true}`
        *   Done:`data: {"done": true, "total": count}`

###`POST /cancel`
Cancels an ongoing stream request.

*   **Request Body:**
```json
    { "id": "stream_uuid" }
    ```
*   **Response:**
```json
    { "success": true }
    ```

###`POST /edit`
Edits a specific message in the current chat history.

*   **Request Body:**
```json
    {
      "index": 0,
      "content": "New message text"
    }
    ```
*   **Response:**
```json
    { "success": true, "total": 5 }
    ```

###`POST /delete`
Deletes a message and all subsequent messages from the history.

*   **Request Body:**
```json
    { "index": 2 }
    ```
*   **Response:**
```json
    { "success": true, "remaining": 2 }
    ```

###`POST /upload`
Uploads a file (image or text) to the chat context.

*   **Request Body:**
```json
    {
      "filename": "image.png",
      "content": "base64_encoded_data",
      "mimetype": "image/png",
      "is_image": true
    }
    ```
*   **Response:**
```json
    { "success": true, "total": 1, "type": "image" }
    ```

## Chat Management

###`GET /chats`
Lists all saved chats.

*   **Response:**
```json
    {
      "chats": [
        {
          "id": "uuid",
          "title": "My Chat",
          "category": "general",
          "tags": ["idea"],
          "created": "2023-10-27T10:00:00",
          "updated": "2023-10-27T11:00:00",
          "message_count": 10,
          "messages": [...preview...]
        }
      ]
    }
    ```

###`GET /chat/load`
Loads a specific chat into the current context.

*   **Query Parameters:**
    *`id` (string): The chat UUID.

*   **Response:**
```json
    {
      "success": true,
      "chat": {
        "id": "uuid",
        "title": "Loaded Chat",
        "messages": [...],
        "total": 5
      }
    }
    ```

###`GET /chat/current`
Returns details of the currently active chat.

*   **Response:** Same structure as`/chat/load`.

###`POST /chat/new`
Creates a new chat context.

*   **Request Body (Optional):**
```json
    { "title": "New Chat", "category": "general" }
    ```
*   **Response:**
```json
    {
      "success": true,
      "chat": { "id": "uuid", "title": "...", "messages": [] }
    }
    ```

###`POST /chat/clear`
Clears messages from the current chat context (keeps the chat container).

*   **Response:**`{"success": true}`

###`POST /chat/delete`
Deletes a saved chat entirely.

*   **Request Body:**
```json
    { "id": "uuid_to_delete" }
    ```
*   **Response:**`{"success": true}`

###`POST /chat/rename`
Renames the current chat.

*   **Request Body:**
```json
    { "title": "New Title" }
    ```
*   **Response:**`{"success": true, "title": "New Title"}`

## Storage Editor (Data Files)

###`GET /storage/list`
Lists editable storage files in the data directory.

*   **Response:**
```json
    {
      "files": [
        { "path": "memory.json", "type": "dict", "name": "memory.json" }
      ],
      "data_dir": "/path/to/data"
    }
    ```

###`GET /storage/load`
Loads content of a specific storage file.

*   **Query Parameters:**
    *`file` (string): Relative path to the file.

*   **Response:**
```json
    {
      "success": true,
      "type": "dict",
      "keys": ["key1", "key2"],
      "data": { "key1": "value1" }
    }
    ```

###`POST /storage/save`
Saves content to a storage file.

*   **Request Body:**
```json
    {
      "file": "memory.json",
      "type": "dict",
      "data": { "key": "updated_value" }
    }
    ```
*   **Response:**`{"success": true}`

###`POST /storage/add-key`
Adds a key to a dictionary-based storage file.

*   **Request Body:**
```json
    { "file": "memory.json", "key": "new_key" }
    ```
*   **Response:**
```json
    { "success": true, "keys": [...], "data": {...} }
    ```

###`POST /storage/delete-key`
Deletes a key from a dictionary-based storage file.

*   **Request Body:**
```json
    { "file": "memory.json", "key": "old_key" }
    ```
*   **Response:**
```json
    { "success": true, "keys": [...], "data": {...} }
    ```
